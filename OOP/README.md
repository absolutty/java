# Objektovo orientované programovanie
**Objekt**:
- základný pojem objektovo orientovaného programovania
- všetky veci okolo nás sú objekty:
  - vec (skriňa, chladnička, ...)
  - človek, študent, ...
- sú dva pohľady na objekty
  - *vonkajší*:
    - k čomu nám slúži?
    - ako sa používa?
  - *vnútorný*:
    - jeho zloženie
    - charakteristika
    - fungovanie
